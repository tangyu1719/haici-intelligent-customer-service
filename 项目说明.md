# 项目说明

## 项目概述

HaiCi 智能客服系统是基于开源电商客服 Agent 项目减重改造的企业级 RAG 问答 MVP，满足 HaiCi 笔试 PRD 对目录结构、前后端分离、流式问答与知识库管理的要求。

## 业务目标

- 为用户提供基于私有知识库的智能问答能力
- 支持多轮会话、意图识别、引用溯源与用户反馈
- 支持知识库文档上传与向量检索

## 技术选型

| 层级 | 技术 |
|------|------|
| 后端框架 | FastAPI |
| 关系数据库 | MySQL 8 |
| 向量数据库 | ChromaDB |
| 大语言模型 | OpenAI 兼容 API（通义千问等） |
| Embedding | BAAI/bge-small-zh-v1.5（CPU） |
| 前端 | Vue 3 + TypeScript + Vite |

## 减重改造说明

相对原仓库已移除：Elasticsearch、Redis、vLLM 本地大模型、GPU Reranker、Agent/MCP 电商工具链、评测面板等重型依赖。

保留并精简：RAG 检索链路、规则意图识别、SSE 流式输出、原 UI 视觉骨架。
