# 运行指南

## 环境要求

- Python 3.10+
- Node.js 18+（前端构建/开发）
- Docker Desktop（MySQL + Chroma）
- LLM API Key（通义千问推荐）

## 步骤

### 1. 环境配置

```bash
cp .env.example .env
# 编辑 .env，至少配置 LLM_API_KEY
```

### 2. 启动中间件

```bash
docker compose up -d
```

MySQL 映射端口 `3307`，Chroma 映射端口 `8001`。

### 3. 初始化数据库

将 `backend/数据库初始化脚本/init.sql` 导入 MySQL（首次部署）。

### 4. 启动后端

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate    # Windows
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 5. 前端

**开发模式**（热更新，API 代理至 8000）：

```bash
cd frontend
npm install
npm run dev
# 访问 http://127.0.0.1:5173
```

**生产模式**（由后端托管静态资源）：

```bash
cd frontend
npm install
npm run build
# 访问 http://127.0.0.1:8000
```

### 6. 验证

1. 注册账号并登录
2. 在「知识库」页上传 txt/md/pdf
3. 在「智能对话」页提问，观察 SSE 流式输出与引用来源

## 代理说明（Git 克隆）

若 GitHub 访问慢，可为 git 设置代理：

```bash
git -c http.proxy=http://127.0.0.1:7897 clone ...
```
